# @sanjo/set

This work is devoted to God.

Functions for sets.

## How to install

```
npm install --save '@sanjo/set'
```

## How to use

```js
import {
  includes,
  union,
  intersection,
  difference,
  symmetricDifference,
  cartesianProduct,
  powerSet,
  isStrictSubset,
  isSubset,
  without
} from "@sanjo/set";
```
