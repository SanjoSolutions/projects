## Package manager

This repository uses [Yarn](https://yarnpkg.com/getting-started/install).

## Creating a new package

```sh
npm init @sanjo/package '<PACKAGE_NAME>' '<PACKAGE_DESCRIPTION>'
```

E.g.

```sh
npm init @sanjo/package '@sanjo/arithmetic' 'Arithmetic'
```
